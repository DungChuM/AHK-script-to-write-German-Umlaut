# AHK-script-to-write-German-Umlaut
Simple AutoHotkey Script to write German umlauts easy (As a Vietnamese that familiar with unikey Telex)

# How to use
What you type -> What character show up
- aw -> ä
- Shift+A w -> Ä
- aww -> aw
<br>

- uw -> ü
- Shift+U w -> Ü
- uww -> uw
<br>

- ow -> ö
- Shift+O w -> Ö
- oww -> ow
<br>

- ss -> ß
- sss -> ss
<br>

- es -> é
- Shift+E s -> É
- ess -> es
<br>
